# Numeric Keypad

Use the standard platform keypad.

## Required Keys
Digits, decimal point, backspace, submit, clear. Level 3 also needs mixed-number entry support for answers such as `8 2/5`.

## Validation
Reject negative values. Accept equivalent decimal forms for decimal questions. For mixed-number questions, accept the intended mixed-number format and optionally equivalent decimal if the template supports it.

## Accessibility
Mirror keypad input through physical keyboard events.
