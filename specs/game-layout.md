# Game Layout - Perimeter Explorer

## Layout Deviations
Use the template shell unchanged. The canvas owns the central experience. The only game-specific HUD is the lap-total panel.

## Question Text Format
Level 1: short, arcade-style prompts.
Level 2: concise maths prompts with diagrams.
Level 3: IXL-style full sentences.

## Progress Config
Use 10 running-shoe icons per level. Correct fills one shoe. Wrong removes at most one shoe for the active question.

## Keypad
Use the standard numeric keypad. Enable decimal input for Levels 2-3 and mixed-number input for Level 3.
